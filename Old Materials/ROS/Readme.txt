Ветка посвящена разработки под ROS.

========================================================
2boffb_node.cpp - в силу того, что до публикации материал закрытый, могу представить в открытом доступе только базовую ноду которая была написана в процессе знакомства с ROS и обучению работе с пакетом PX4-LIRS. Нода представляет из себя код, который заставляет дрон взлететь, и перелететь по заданны координатам. 
========================================================
rqt_gui_main_cpp - на основе общей работы была проведена индивидуальная разработка проекта ответвления для выступления на ICCT 2022. Графический интерфейс для управления дроном в симуляции, вывода изображения с камеры запрограммированой на основе ESP32-CAM с использованием openCV, вывода изображения из симуляции, вывода информации из топиков в rviz-display плагин внутри общего GUI.
