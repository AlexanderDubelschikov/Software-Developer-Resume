Разработка под плату umdk-RF из под линукс в RIOT OS.

prog1 - создание переменных и проверка, в какой части памяти они сохраняются.
========================================================
prog2.4 - прерывание. Различные темп моргания светодиодом на плате с помощью прерывания. 
========================================================
prog3 - инверсия приоритетов. Демонстрация с помощью мьютексов. 
========================================================
prog4.2 - устройство кодового сейфа с использованием четырехкнопочной клавиатуры. 
========================================================
prog5 - светофор, с возможностью ускорить наступление зеленого света по нажатию кнопки.
========================================================
prog6 - вывод информации с датчиков. 
========================================================
